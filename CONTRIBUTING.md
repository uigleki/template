# Contributing

✨ Every contribution makes a difference!

## 🌈 Dev Environment

**🚀 GitHub Codespaces**

- Browser or VS Code
- 60 free hours/month
- Startup takes time
- Network dependent

**💻 Local DevContainer**

- Docker required
- Large disk space
- Works offline

**🔧 Manual Setup**

- Check `.devcontainer`

## ⚡️ Quick Start

```bash
# 1. Fork & Clone
git clone https://github.com/your-username/project.git

# 2. Start Environment
# - Codespaces
# - VS Code + DevContainer
# - Manual setup

# 3. Create Branch
git checkout -b feat/amazing

# 4. Make Changes ✨

# 5. Submit PR 🎉
```
